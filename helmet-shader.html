▶ Background


An interactive 3D dithering shader implementation showcasing real-time graphics programming. The project features a Supreme Fox racing helmet with customizable pixelation effects, halftoning patterns, and bloom post-processing. Built with React Three Fiber, it demonstrates advanced shader techniques including 4x4 dithering matrices, luminance-based pattern application, and dynamic parameter controls.
